# Population by Postal Code Area

A sample of population data by postal code area from Statistics
Finland's Paavo statistics table.

## Usage

``` r
vaesto_postinumeroittain
```

## Format

### `vaesto_postinumeroittain`

Data frame with 886 rows and 6 columns:

- geom:

  Multipolygon object

- Postinumero:

  Postal Code

- Postinumeroalue:

  Name of the Postal Code Area

- Alue:

  Municipality of the Postal Code Area

- time:

  Year

- value:

  Population
