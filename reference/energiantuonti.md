# Energy Import and Export by Country of Origin

A sample of energy imports and exports by country of origin from
Statistics Finland's archive table.

## Usage

``` r
energiantuonti
```

## Format

### `energiantuonti`

Data frame with 723 rows and 4 columns:

- Alue:

  The name of the country or country group

- Suunta:

  Import or export

- time:

  Quarter

- value:

  Value (million euros)
