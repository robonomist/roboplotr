# Error bar configuration

Parameters to add and customize error bars in
[roboplots](https://robonomist.github.io/roboplotr/reference/roboplot.md).

## Usage

``` r
set_errorbars(...)
```

## Arguments

- ...:

  Parameters passed to
  [`set_confidence_interval()`](https://robonomist.github.io/roboplotr/reference/set_confidence_interval.md).

## Value

A list of class roboplot.set_confidence_interval
