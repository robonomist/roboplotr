# Package index

## Main plot functions

Build basic plots

- [`roboplot()`](https://robonomist.github.io/roboplotr/reference/roboplot.md)
  : Comprehensive plotly wrapper function
- [`robotable()`](https://robonomist.github.io/roboplotr/reference/robotable.md)
  : Comprehensive DT wrapper function
- [`robomap()`](https://robonomist.github.io/roboplotr/reference/robomap.md)
  : Comprehensive leaflet wrapper function

## Set plotting parameters

Helper functions to change the appearance of plots

- [`set_roboplot_options()`](https://robonomist.github.io/roboplotr/reference/set_roboplot_options.md)
  :

  Global configuration of various `roboplotr` outputs

- [`set_artefacts()`](https://robonomist.github.io/roboplotr/reference/set_artefacts.md)
  : Visualization export configuration

- [`set_axes()`](https://robonomist.github.io/roboplotr/reference/set_axes.md)
  : Axis configuration

- [`set_border()`](https://robonomist.github.io/roboplotr/reference/set_border.md)
  : Plot border configuration

- [`set_caption()`](https://robonomist.github.io/roboplotr/reference/set_caption.md)
  : Caption configuration

- [`set_confidence_interval()`](https://robonomist.github.io/roboplotr/reference/set_confidence_interval.md)
  : Configure Confidence Display

- [`set_empty_roboplot()`](https://robonomist.github.io/roboplotr/reference/set_empty_roboplot.md)
  :

  Empty
  [`roboplot()`](https://robonomist.github.io/roboplotr/reference/roboplot.md)
  configuration

- [`set_errorbars()`](https://robonomist.github.io/roboplotr/reference/set_errorbars.md)
  : Error bar configuration

- [`set_externalmenu()`](https://robonomist.github.io/roboplotr/reference/set_externalmenu.md)
  : Set up an external menu for filtering plot data based on a
  categorical variable. This function creates a configuration for an
  external menu that can be used to filter plot data based on a
  categorical variable. The menu will be rendered as a draggable overlay
  on the plot, with checkboxes for each unique value of the specified
  variable. The menu can be toggled from the modebar.

- [`set_font()`](https://robonomist.github.io/roboplotr/reference/set_font.md)

- [`set_grid()`](https://robonomist.github.io/roboplotr/reference/set_grid.md)

- [`set_heatmap()`](https://robonomist.github.io/roboplotr/reference/set_heatmap.md)

- [`set_hovertext()`](https://robonomist.github.io/roboplotr/reference/set_hovertext.md)
